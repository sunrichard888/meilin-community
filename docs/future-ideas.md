# Future Ideas

Parking lot for out-of-scope ideas, future enhancements, and aspirational features. These are not committed work — just ideas to revisit.

## Product Ideas

- **Neighborhood Groups**: Sub-communities for specific buildings, streets, or interest groups
- **Events Calendar**: Community events, meetups, garage sales
- **Buy/Sell Marketplace**: Local classifieds for neighbors
- **Recommendation System**: "Best plumber in the neighborhood" crowdsourced recommendations
- **Safety Alerts**: Amber alerts, suspicious activity warnings
- **Lost & Found**: Dedicated category for lost pets, found items
- **Welcome Committee**: Automated welcome posts for new residents
- **Local Business Directory**: Verified local businesses, special offers for neighbors
- **Polls & Surveys**: Community decision-making tools
- **Resource Sharing**: Tool library, carpooling, skill sharing

## Technical Ideas

- **Real-time Updates**: WebSocket/pusher for live feed updates
- **PWA Support**: Install as app, offline reading
- **Push Notifications**: Browser push for new activity
- **Image Optimization**: Auto-resize, compress, CDN delivery
- **Search**: Full-text search across posts and comments
- **Export Data**: Let users download their data (GDPR compliance)
- **Dark Mode**: User preference for dark theme
- **Internationalization**: Support multiple languages beyond Chinese

## Growth Ideas

- **Referral Program**: Invite neighbors, unlock features
- **Verified Addresses**: Mail verification for trust badges
- **Partner with Property Management**: Building-wide adoption
- **Local SEO**: Appear in "neighborhood near me" searches
- **Social Sharing**: Share posts to WeChat, Weibo
- **Community Ambassadors**: Power users who onboard neighbors

## Metrics to Track Post-Launch

- DAU/WAU ratio (engagement health)
- Posts per active user per week
- Comment-to-post ratio (conversation depth)
- User retention D1, D7, D30
- Time to first post (onboarding friction)
- Report rate (community health indicator)
- Mobile vs desktop usage
