# 侧边栏页面开发计划

## 当前问题

首页右侧侧边栏有以下模块，但链接无法访问：
1. **🔥 热门话题** - 链接到 `/hot-topics`
2. **📢 社区公告** - 链接到 `/announcements`
3. **📊 社区概况** - 链接到 `/community-stats`

---

## 开发计划

### 任务 1: 热门话题页面
**优先级**: 高  
**工作量**: 30 min

**功能**:
- 热门话题列表（按帖子数/参与度排序）
- 话题标签云
- 点击话题筛选帖子

**页面结构**:
```
/hot-topics
├── 话题列表
│   ├── 话题名称
│   ├── 参与人数
│   └── 帖子数
└── 相关帖子推荐
```

### 任务 2: 社区公告页面
**优先级**: 高  
**工作量**: 30 min

**功能**:
- 官方公告列表
- 置顶公告
- 公告详情

**页面结构**:
```
/announcements
├── 公告列表
│   ├── 标题
│   ├── 发布时间
│   └── 发布者（管理员）
└── 公告详情（点击展开）
```

**数据库表**:
```sql
CREATE TABLE announcements (
  id UUID PRIMARY KEY,
  title VARCHAR(200),
  content TEXT,
  is_pinned BOOLEAN DEFAULT FALSE,
  published_at TIMESTAMP,
  created_by UUID REFERENCES users(id),
  created_at TIMESTAMP
);
```

### 任务 3: 社区概况页面
**优先级**: 中  
**工作量**: 20 min

**功能**:
- 社区统计数据
- 用户数
- 帖子数
- 今日活跃用户
- 热门小区

**页面结构**:
```
/community-stats
├── 统计卡片
│   ├── 总用户数
│   ├── 总帖子数
│   ├── 今日活跃
│   └── 小区数量
├── 热门小区排行
└── 活跃用户排行
```
