# P3 后台管理系统 - 任务清单

## 目标
完成美邻网后台管理系统，提供管理员运营工具。

---

## 功能 1: 管理员仪表盘

**URL**: `/admin/dashboard`

**需求**:
- [ ] 用户总数/今日新增
- [ ] 帖子总数/今日发帖
- [ ] 举报待处理数
- [ ] 活跃用户趋势（7 天）
- [ ] 热门帖子排行

---

## 功能 2: 用户管理

**URL**: `/admin/users`

**需求**:
- [ ] 用户列表（分页/搜索）
- [ ] 用户详情
- [ ] 封禁/解封用户
- [ ] 修改用户角色（user/admin）
- [ ] 删除用户

---

## 功能 3: 内容审核

**URL**: `/admin/moderation`

**需求**:
- [ ] 举报列表
- [ ] 处理举报（删除/忽略）
- [ ] 帖子管理（删除/置顶）
- [ ] 批量操作

---

## 功能 4: 公告管理

**URL**: `/admin/announcements`

**需求**:
- [ ] 公告列表
- [ ] 发布新公告
- [ ] 编辑公告
- [ ] 删除公告
- [ ] 置顶/取消置顶

---

## 实施顺序

1. **管理员仪表盘** - 最简单，纯展示
2. **公告管理** - CRUD 操作
3. **用户管理** - 需要权限控制
4. **内容审核** - 最复杂

---

## 权限控制

- 所有 `/admin/*` 页面需要 admin 角色
- 前端路由守卫
- 后端 API 验证
